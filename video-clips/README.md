# Video-Clips für Creator

Dieser Ordner enthält herunterladbare Video-Clips für TikTok-Creator.

## Verwendung

1. Lade deine Video-Clips (.mp4) in diesen Ordner hoch
2. Die Videos werden automatisch auf der Website im "Video-Clips" Tab angezeigt
3. Creator können die Videos direkt herunterladen und in ihren TikToks verwenden

## Format

- Dateiformat: MP4
- Empfohlene Auflösung: 1080x1920 (TikTok Format)
- Maximale Dateigröße: 100MB pro Video

## Struktur

Jedes Video sollte folgende Informationen haben:
- Titel/Beschreibung
- Thumbnail (wird automatisch generiert)
- Download-Link
